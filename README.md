# Profile Page 2

This is a modern portfolio website built with Next.js, TypeScript, and Tailwind CSS. It features:
- A homepage with hero section
- About Me
- Professional highlights
- Personal interests
- Featured projects
- Contact & social links
- Footer

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Visit `http://localhost:3000` in your browser.

## Customization
- Add your content and images to the relevant sections in `src/app`.
- Expand with blog, gallery, or other features as needed.

---

For more details, see the Next.js and Tailwind CSS documentation.
